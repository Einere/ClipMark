# Markdown Quartz: Full Design System Specification (Ready for AI Implementation) 이 문서는 **Markdown Quartz** 디자인 시스템의 모든 시각적 규칙과 기술적 명세를 담고 있습니다. 이 내용을 기반으로 React, Tailwind CSS, 또는 기타 프레임워크를 사용해 일관된 UI를 구현할 수 있습니다. --- ## 1. Overview & Creative North Star **"The Digital Atheneum"** 정갈함, 고전적인 신뢰감, 그리고 현대적인 미니멀리즘이 공존하는 macOS 중심의 텍스트 에디터 디자인 시스템입니다. '글쓰기'라는 본질에 집중할 수 있도록 시각적 노이즈를 최소화하고 깊이 있는 레이어링을 사용합니다. --- ## 2. Color Palette (Design Tokens) 모든 컬러는 하드코딩된 HEX 값 대신 아래의 토큰 이름을 사용하여 구현해야 합니다. ### Core Branding & Surface | Token | Hex | Description / Usage | | :--- | :--- | :--- | | `primary` | `#0366D6` | 핵심 CTA, 주요 하이라이트, 포인트 컬러 (Quartz Blue) | | `secondary` | `#24292F` | 보조 액션, 깊이감 있는 텍스트, 다크 모드 대비 (Charcoal) | | `tertiary` | `#4078C0` | 강조 텍스트, 링크, 보조 하이라이트 (Accent Blue) | | `neutral` | `#F6F8FA` | 배경색, 섹션 구분, 중립적인 영역 (Light Gray) | | `surface-snow` | `#F8F9FB` | 에디터 및 콘텐츠 패널 배경 | | `surface-ghost` | `#FFFFFF` | 메인 배경, 카드 배경 (Pure White) | | `surface-secondary`| `#F2F4F6` | 사이드바 배경, 미묘한 섹션 구분 (Soft Gray) | | `stroke-mist` | `#E1E4E8` | 일관된 구분선, 테두리 컬러 | ### Text Hierarchy | Token | Hex | Usage | | :--- | :--- | :--- | | `text-primary` | `#191C1E` | 헤드라인, 메인 본문 텍스트 (Deep Dark) | | `text-secondary` | `#5A5E62` | 부제목, 캡션, 플레이스홀더 (Medium Gray) | | `text-tertiary` | `#A0A4A8` | 메타데이터, 푸터 링크, 비활성 상태 (Light Gray) | --- ## 3. Typography Hierarchy **UI Font Family:** `Inter`, `-apple-system`, `sans-serif` **Editor Font Family:** `SF Mono`, `JetBrains Mono`, `monospace` | Level | Size | Weight | Tracking | Color | | :--- | :--- | :--- | :--- | :--- | | **H1 (Hero)** | 72px | Bold | -2% | `text-primary` | | **H2 (Section)** | 32px | Bold | -1% | `text-primary` | | **H3 (Panel)** | 18px | Semibold | -0.5% | `text-primary` | | **H4 (Sub-text)** | 14px | Bold | 5% (Upper) | `text-secondary` | | **Body Large** | 18px | Regular | 0 | `text-primary` | | **Body Base** | 14px | Regular | 0 | `text-primary` | | **UI Small** | 12px | Medium | 0 | `text-secondary` | --- ## 4. Spacing & Gutter (The 8pt Grid) 시스템의 모든 간격은 8px 단위를 기준으로 하며, 일관된 리듬을 유지합니다. * **Base Unit:** 8px * **Small (S):** 8px / 12px (컴포넌트 내부 간격) * **Medium (M):** 16px / 24px (섹션 내 요소 간격) * **Large (L):** 32px / 48px (패널 간 간격, 히어로 섹션 패딩) * **Gutter:** 1px (패널 구분선 두께) --- ## 5. Common UI Components ### Buttons * **Primary Button:** `bg-primary`, `text-white`, `rounded-lg (8px)`, Padding: `px-6 py-3`. * **Secondary Button:** `bg-surface-secondary`, `text-text-secondary`, `rounded-lg (8px)`. * **Icon Button:** `bg-transparent`, `hover:bg-surface-secondary`, `rounded-full`, 24x24px icon. ### Cards & Panels * **File Card (Recent):** `bg-surface-ghost`, `hover:shadow-md`, `rounded-xl (12px)`, Padding: `16px`. * **Editor Panel:** `bg-surface-snow`, `border-l border-stroke-mist`. * **Status Bar:** `h-10`, `bg-surface-secondary`, `text-text-tertiary`, Left-aligned text. ### Dividers * **Vertical/Horizontal:** 1px, `bg-stroke-mist`. --- ## 6. Iconography Style * **Style:** Outlined, 1.5px Stroke weight. * **Size:** Base 24x24px. * **Color:** `text-text-secondary`. --- ## 7. Implementation Code Snippet (Tailwind Config Example) ```javascript module.exports = { theme: { extend: { colors: { primary: '#0366D6', secondary: '#24292F', tertiary: '#4078C0', neutral: '#F6F8FA', surface: { snow: '#F8F9FB', ghost: '#FFFFFF', secondary: '#F2F4F6', }, stroke: { mist: '#E1E4E8', }, text: { primary: '#191C1E', secondary: '#5A5E62', tertiary: '#A0A4A8', } }, fontFamily: { sans: ['Inter', 'sans-serif'], mono: ['SF Mono', 'monospace'], }, borderRadius: { 'lg': '8px', 'xl': '12px', } } } } ```
