# Markdown Quartz Design System Specification (Unified) ## 1. Overview & Creative North Star **"The Digital Atheneum"** Markdown Quartz는 정갈함, 고전적인 신뢰감, 그리고 현대적인 미니멀리즘이 공존하는 macOS 중심의 텍스트 에디터 디자인 시스템입니다. 복잡한 기능을 걷어내고 '글쓰기'라는 본질적인 행위에만 집중할 수 있는 환경을 제공합니다. --- ## 2. Design Tokens: Color Palette 시스템 전반에서 일관성을 유지하기 위해 정의된 핵심 컬러 토큰입니다. ### Core Branding & Surface Tokens | Token Name | Hex Value | Usage / Description | | :--- | :--- | :--- | | `primary` | `#0366D6` | 핵심 CTA, 주요 하이라이트, 포인트 컬러 (Quartz Blue) | | `secondary` | `#24292F` | 보조 액션, 깊이감 있는 텍스트, 다크 모드 대비 (Charcoal) | | `tertiary` | `#4078C0` | 강조 텍스트, 링크, 보조 하이라이트 (Accent Blue) | | `neutral` | `#F6F8FA` | 배경색, 섹션 구분, 중립적인 영역 (Light Gray) | | `surface-snow` | `#F8F9FB` | 에디터 및 콘텐츠 패널 배경 | | `surface-ghost` | `#FFFFFF` | 메인 배경, 카드 배경 (Pure White) | | `surface-secondary`| `#F2F4F6` | 사이드바 배경, 미묘한 섹션 구분 (Soft Gray) | | `stroke-mist` | `#E1E4E8` | 일관된 구분선, 테두리 컬러 | ### Text Hierarchy Tokens | Token Name | Hex Value | Usage / Description | | :--- | :--- | :--- | | `text-primary` | `#191C1E` | 헤드라인, 메인 본문 텍스트 (Deep Dark) | | `text-secondary` | `#5A5E62` | 부제목, 캡션, 플레이스홀더 (Medium Gray) | | `text-tertiary` | `#A0A4A8` | 메타데이터, 푸터 링크, 비활성 상태 (Light Gray) | --- ## 3. Design Tokens: Typography **Font Family:** UI 요소에는 `Inter` (Sans-serif), 에디터 코드 블록 및 본문에는 `SF Mono` 또는 `JetBrains Mono`를 사용합니다. ### Headlines * **H1 (Hero Title):** 72px / Bold / Tracking -2% / `text-primary` * **H2 (Section Header):** 32px / Bold / Tracking -1% / `text-primary` * **H3 (Panel Header):** 18px / Semibold / Tracking -0.5% / `text-primary` * **H4 (Sub-section):** 14px / Bold / Uppercase / Tracking 5% / `text-secondary` ### Body & UI Text * **Body Large:** 18px / Regular / Leading 1.6 / `text-primary` * **Body Base:** 14px / Regular / Leading 1.5 / `text-primary` * **UI Small (Metadata):** 12px / Medium / `text-secondary` * **Code (Editor):** 14px / Regular / Leading 1.6 / `SF Mono` --- ## 4. Common UI Components ### Buttons * **Primary Button:** Background `primary`, White text, 8px Border Radius. * **Secondary Button:** Background `secondary`, White text, 8px Border Radius. * **Neutral Button:** Background `neutral`, `text-secondary`, 8px Border Radius. * **Ghost/Icon Button:** Transparent background, Hover state with `surface-secondary` circle. ### Cards & Panels * **File Card (Landing):** `surface-ghost` background, No border, Subtle shadow on hover. * **Editor Panel:** Background `surface-snow`, Vertical divider `stroke-mist`. * **TOC Panel (Contents):** Background `surface-secondary`, Indented hierarchy. ### Layout Constants * **Dividers:** 1px width, Color `stroke-mist`. * **Border Radius:** 8px (Standard), 12px (Cards). * **Gutter/Spacing:** Base 8px (Multiples: 16px, 24px, 32px, 48px). --- ## 5. Iconography **Style:** Outlined, 1.5px Stroke weight, Optically balanced for 24x24px. * **Settings:** `settings` (Material Symbols) * **New File:** `add_circle` * **Open Folder:** `folder_open` --- ## 6. Markdown Specifics (Editor) * **# H1:** Bold, `primary` accent on the `#` symbol. * **## H2:** Bold, slightly smaller than H1. * **Blockquotes:** 3px left border `primary`, Italic text. * **Code Blocks:** Background `surface-secondary`, Monospace font.
